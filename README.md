BackendAI is a frontend web application designed to provide an intuitive interface for interacting with AI services. The project demonstrates modern web development practices with clean, organized code structure.

## ✨ Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Interactive UI** - Dynamic JavaScript functionality for enhanced user experience
- **Modern Styling** - Clean and professional CSS styling
- **Easy to Use** - Simple and intuitive interface

## 🗂️ Project Structure

```
backendai/
├── index.html       # Main HTML file with page structure
├── style.css        # CSS styling and layouts
```

## 📁 File Descriptions

- **index.html** - The main HTML file containing the page structure and markup. This is the entry point of the application.
- **style.css** - Comprehensive CSS file that handles all styling, layout, colors, fonts, and responsive design.
- **script.js** - JavaScript file containing all client-side logic, event handlers, and DOM manipulation.
- **README.md** - Project documentation and setup guide.

## 🚀 Installation

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Git installed on your system
- Internet connection (for initial setup)

### Steps

1. **Clone the repository**
```bash
git clone https://github.com/harmanjotsha/backendai.git
```

2. **Navigate to the project directory**
```bash
cd backendai
```

3. **Open the application**
   - Simply open `index.html` in your web browser
   - Or use a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Or using Node.js with http-server
   npx http-server
   ```

## 💻 Usage

1. Open `index.html` in your web browser
2. Interact with the application through the user interface
3. All functionality is client-side, no backend server required for basic operation
4. Check the browser console (F12) for any debugging information

## 📦 Requirements
